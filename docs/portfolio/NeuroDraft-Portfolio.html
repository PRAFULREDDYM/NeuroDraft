THE EXECUTION

Six agents. One result. Thirty seconds.

The pipeline is built on Next.js 15 App Router with six parallel and sequential Groq API calls, Server-Sent Events for real-time streaming, and a Zustand store that syncs every agent's state to the frontend as it changes.

The neural scoring system is calibrated against TRIBE v2's published behavior patterns — specifically the relationships between stimulus properties (humor, surprise, emotional intensity, visual complexity) and which brain regions activate. An independent evaluator agent validates every output before it reaches the user.
